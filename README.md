# Note Vault

以目标为驱动，把零散阅读重点串联成知识脉络的系统。

## 功能

- **目标管理**：创建学习目标，拆解为子问题
- **文章来源**：管理阅读材料，标记阅读状态
- **重点卡片**：从文章摘录重点，强制关联到子问题
- **卡片关联**：支持"支撑/补充/矛盾/递进"语义关联
- **知识图谱**：D3.js 力导向图，可视化知识网络
- **组合检索**：按目标、标签、关键词检索卡片
- **进度追踪**：自动计算子问题覆盖率
- **导入导出**：JSON 备份 / Markdown 导出

## 使用

直接在浏览器打开 `index.html` 即可，数据存储在浏览器 localStorage。

## 部署

纯静态文件，可直接托管到 GitHub Pages、Vercel 等平台。
